# UX Profile: deck_full

## Screen Class

Keyboard handheld with medium display.

## Input Model

Keyboard, directional controls, optional touch.

## Feature Set

Full field workflow: chat, map, team actions, GPS, settings, tracker, and
device status.

## Screen Set

Dashboard, Chat, Team, Map/GPS, Tracker, Settings, Contacts, Extensions.

## Map Mode

Full map mode with pan/zoom, offline tiles, current position, and overlay
snapshots.

## Chat Mode

Full chat mode with conversation list, compose, delivery state, key
verification, and Team rich payload summaries.

## Team Action Mode

Full Team action mode with location marker picker and command workflows.

## GPS Mode

Full GPS status and map-adjacent GPS workflow.

## Modal/Picker Strategy

Keyboard-friendly modal and picker renderers from the selected UX pack.

## Renderer Family

LVGL.

## Deferred Decisions

Final screen density, keyboard shortcuts, and rich Team card renderer remain
deferred until UX pack implementation.

Board describes. Target chooses. UX Pack presents. Renderer draws.
